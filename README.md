# Manoj-Technologies
A modern, responsive corporate website for Manoj Technologies, built using pure HTML, CSS, and JavaScript. This website showcases company services, expertise, business metrics, and includes an interactive contact form with smooth scrolling navigation.

# Features

Fully responsive modern UI

Sticky navigation with smooth scroll

Hero section with strong branding

Business metrics showcase

Services & expertise sections

Professional CTA (Call to Action)

Interactive “Let’s Connect” contact form

Clean and structured single-file implementation

No external frameworks required
